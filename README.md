# financial-analyzer

A real-time financial dashboard for SPX market analysis, options analytics, and AI-generated market reports. Built for single-operator use with a focus on options market intelligence.

## Quickstart

```bash
# 1. Clone and install
git clone <repo-url> ~/repos/financial-analyzer
cd ~/repos/financial-analyzer
npm install

# 2. Configure environment
cp .env.example .env.local  # or create manually
# Set: ANTHROPIC_API_KEY=sk-ant-...
# Set: REPORT_SECRET=some-random-secret

# 3. Start the dev server
npm run dev -- --port 3001

# 4. Open the dashboard
open http://localhost:3001
```

The database (`data/reports.db`) is created automatically on first startup.

## Features

- **SPX market charts** — live price data from Yahoo Finance and FRED, with configurable time ranges (1D, 5D, 1M, 3M, 6M, 1Y, YTD)
- **Options analytics** — IV, Greeks (delta, gamma, vega, theta), volatility regime, skew analysis
- **Options price overlay** — chart option contract prices alongside underlying price history
- **Probability projections** — expected price distributions and key support/resistance levels
- **AI market reports** — daily morning/midday/EOD reports generated by Claude with market data context
- **Fear & Greed index** — CNN Fear & Greed proxy with historical comparison
- **Reports dashboard** — browse, compare, and read past AI-generated market reports

## Technology Stack

| Technology | Purpose |
|---|---|
| **Next.js 16** (App Router) | Framework — routing, SSR, API routes |
| **React 19** | UI components |
| **TypeScript** (strict) | Type safety |
| **SQLite** (better-sqlite3) | Persistent storage — reports, option data |
| **Tailwind CSS v4** | Styling (dark-only design) |
| **lightweight-charts** | Financial chart rendering |
| **Anthropic Claude** | AI report generation and options analysis |
| **Vitest** | Unit and integration tests |
| **Playwright** | End-to-end tests |
